⌨️ 快速鍵與操作說明

滑鼠操作

  • 右鍵 點元件或空白處 → 操作選單
  • 雙擊 課程類別卡 → 開啟班名清單
  • 雙擊 文字／色塊 → 直接編輯文字
  • 空白處左鍵拖曳 → 框選多個元件
  • Shift + 左鍵 → 累加選取
  • Space + 拖曳 或 滑鼠中鍵 + 拖曳 → 平移畫布
  • Ctrl + 滑鼠滾輪 → 縮放

鍵盤快速鍵

  • Esc → 取消連線模式／取消選取
  • Delete / Backspace → 刪除選取
  • Ctrl + A → 全選元件
  • Ctrl + D → 複製選取
  • Ctrl + C / V → 複製／貼上
  • Ctrl + G → 建立群組
  • Ctrl + Shift + G → 解除群組

群組與多選

  • 智慧整理後會自動將每個主分類與其子分類群組化
  • 點擊群組中任一元件 → 整組會以 橘色 高亮
  • 拖曳主分類（不論有無群組）→ 所有下游子分類一起平移
